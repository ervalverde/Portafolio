# 📊 Análisis de Ventas - Proyecto de Portafolio

## Descripción
Este proyecto consiste en un análisis exploratorio de datos de ventas utilizando Python. 
El objetivo es identificar patrones de compra, productos más vendidos y tendencias por región.

## Herramientas utilizadas
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
El dataset contiene información de ventas con las siguientes columnas:
- fecha
- producto
- provincia
- monto

## Análisis realizados
- Ventas totales por provincia
- Producto más vendido
- Tendencia de ventas en el tiempo
- Ticket promedio

## Conclusiones
Este análisis permite entender mejor el comportamiento de los clientes y apoyar la toma de decisiones basada en datos.
