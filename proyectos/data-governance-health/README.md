# Trusted Health Data: Proyecto de Gobernanza y Calidad de Datos

Este proyecto de portafolio demuestra cómo aplicar principios de gobernanza de datos sobre un dataset global de salud.

## Objetivos
- Evaluar la calidad de los datos  
- Estandarizar estructuras y nombres  
- Definir y validar reglas de negocio  
- Generar un dataset confiable  
- Construir visualizaciones basadas en datos gobernados  

## Estructura del proyecto
- `data/raw/`: datos originales  
- `data/processed/`: datos procesados  
- `notebooks/`: análisis en Jupyter  
- `docs/`: documentación de gobernanza  
- `src/`: scripts reutilizables  

## Componentes de gobernanza
- Perfilado de datos  
- Evaluación de calidad  
- Reglas de negocio  
- Estandarización  
- Linaje de datos  
- Catálogo de datos  

## Tecnologías utilizadas
- Python  
- Pandas  
- Matplotlib  
- Seaborn  
- Jupyter Notebook
